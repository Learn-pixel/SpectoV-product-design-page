# SpectoV Product Design Page

This is a responsive webpage designed for DefXV AR/VR glasses product using HTML, CSS, and JavaScript.

## Features
- Clean, modern UI
- Brand colors: grey, black, blue, and green
- Countdown timer
- Cards for pricing, user review etc.
- FAQ section
- CTA and footer
